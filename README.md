## Installation

```bash

git clone https://github.com/yourusername/VR-DAW-1.git
cd VR-DAW-1

# Build-Verzeichnis erstellen
mkdir build && cd build

# Projekt konfigurieren
cmake ..

# Projekt bauen
make -j4
```
